# TypeScript の基本 - 型の力を理解しよう

プログラミング初心者の皆さん、**TypeScript**という名前を聞いたことはありますか？TypeScriptは、JavaScriptを拡張した言語で、現在多くの企業や開発者に愛用されています。今回は、TypeScriptの基本と「**型**」の重要性について、初心者にも分かりやすく解説します。

### JavaScriptとTypeScriptの違い

**JavaScript**（従来の書き方）：
```javascript
function greet(name) {
  return "こんにちは、" + name + "さん！";
}

greet("田中");        // "こんにちは、田中さん！"
greet(123);          // "こんにちは、123さん！" ← おかしい...
greet();             // "こんにちは、undefinedさん！" ← エラー！
```

**TypeScript**（型を指定した書き方）：
```typescript
function greet(name: string): string {
  return "こんにちは、" + name + "さん！";
}

greet("田中");        // ✅ 正常
greet(123);          // ❌ エラー：数値は渡せません
greet();             // ❌ エラー：引数が必要です
```

## TypeScriptの基本的な型

### 1. 基本的な型

```typescript
// 文字列型
let name: string = "田中太郎";
let message: string = "こんにちは";

// 数値型
let age: number = 25;
let price: number = 1500;

// 真偽値型
let isStudent: boolean = true;
let hasLicense: boolean = false;

// 配列型
let fruits: string[] = ["りんご", "バナナ", "オレンジ"];
let scores: number[] = [85, 92, 78, 96];
```

### 2. オブジェクト型

実際のアプリケーションでよく使われるオブジェクト型を見てみましょう：

## 実践例：ショッピングカート

実際のWebアプリケーションで使われそうな例を見てみましょう：

```typescript
// 商品の型定義
interface Product {
  id: number;
  name: string;
  price: number;
  inStock: boolean;
}

// カートアイテムの型定義
interface CartItem {
  product: Product;
  quantity: number;
}

// ショッピングカートの関数
function addToCart(product: Product, quantity: number): CartItem {
  if (quantity <= 0) {
    throw new Error("数量は1以上である必要があります");
  }
  
  if (!product.inStock) {
    throw new Error("この商品は在庫切れです");
  }
  
  return {
    product: product,
    quantity: quantity
  };
}

// 使用例
const laptop: Product = {
  id: 1,
  name: "ノートパソコン",
  price: 89800,
  inStock: true
};

const cartItem = addToCart(laptop, 1);
```

## 実際のエラー例とその修正

**エラーが出るコード：**
```typescript
function calculateTotal(items) {  // ← 型がない
  return items.reduce((total, item) => {
    return total + item.price * item.quantity;  // ← エラーの可能性
  }, 0);
}
```

**修正後のコード：**
```typescript
interface CartItem {
  price: number;
  quantity: number;
}

function calculateTotal(items: CartItem[]): number {
  return items.reduce((total: number, item: CartItem) => {
    return total + item.price * item.quantity;
  }, 0);
}
```

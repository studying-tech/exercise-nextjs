# React入門 - コンポーネントとは何か？

プログラミングを始めたばかりの皆さん、**React**という言葉を聞いたことはありますか？Reactは現在最も人気のあるWebアプリケーション開発ライブラリの一つです。今回は、Reactの最も重要な概念である「**コンポーネント**」について、初心者にも分かりやすく解説します。

### 従来のHTMLとReactの違い

従来のHTMLでは、すべてのコードを1つのファイルに書いていました：

```html
<!DOCTYPE html>
<html>
<body>
  <div class="header">
    <h1>私のWebサイト</h1>
    <nav>
      <ul>
        <li><a href="/">ホーム</a></li>
        <li><a href="/about">について</a></li>
      </ul>
    </nav>
  </div>
  
  <div class="content">
    <h2>ようこそ！</h2>
    <p>これは私のWebサイトです。</p>
  </div>
  
  <div class="footer">
    <p>&copy; 2025 私のWebサイト</p>
  </div>
</body>
</html>
```

Reactでは、これを**小さな部品（コンポーネント）**に分けて管理します：

```javascript
// ヘッダーコンポーネント
function Header() {
  return (
    <div className="header">
      <h1>私のWebサイト</h1>
      <nav>
        <ul>
          <li><a href="/">ホーム</a></li>
          <li><a href="/about">について</a></li>
        </ul>
      </nav>
    </div>
  );
}

// メインコンテンツコンポーネント
function MainContent() {
  return (
    <div className="content">
      <h2>ようこそ！</h2>
      <p>これは私のWebサイトです。</p>
    </div>
  );
}

// フッターコンポーネント
function Footer() {
  return (
    <div className="footer">
      <p>&copy; 2025 私のWebサイト</p>
    </div>
  );
}
```

## Props（プロパティ）とは？

コンポーネントに**情報を渡す仕組み**をProps（プロパティ）と呼びます。これも身近な例で説明しましょう。

<div className="bg-orange-50 p-6 rounded-lg border border-orange-200 my-6">
  <h4 className="text-lg font-semibold text-orange-900 mb-3">🏭 工場の例で理解しよう</h4>
  <div className="text-orange-800">
    <strong>コンポーネント = 商品を作る工場</strong><br/>
    <strong>Props = 工場に送る材料</strong><br/><br/>
    同じ工場（コンポーネント）でも、違う材料（Props）を送ると、違う商品ができます！
  </div>
</div>

### Propsを使ったコンポーネント例

```javascript
// 汎用的なボタンコンポーネント
function Button(props) {
  return (
    <button className={`px-4 py-2 rounded ${props.color} text-white`}>
      {props.text}
    </button>
  );
}

// 使用例
<Button text="保存する" color="bg-blue-500" />
<Button text="削除する" color="bg-red-500" />
<Button text="キャンセル" color="bg-gray-500" />
```

実際の表示：

<div className="flex space-x-3 my-4">
  <button className="px-4 py-2 rounded bg-blue-500 text-white">保存する</button>
  <button className="px-4 py-2 rounded bg-red-500 text-white">削除する</button>
  <button className="px-4 py-2 rounded bg-gray-500 text-white">キャンセル</button>
</div>

## まとめ

**Reactコンポーネント**は、Webアプリケーションを**小さな部品に分けて管理する仕組み**です。レゴブロックのように組み合わせることで、複雑なアプリケーションも効率的に作ることができます。

最初は難しく感じるかもしれませんが、少しずつ慣れていけば、きっとReactの便利さを実感できるはずです。頑張ってください！
